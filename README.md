# KLIPY Messenger Extension

A Chrome extension for sending **GIFs, stickers, and clips** in chat using the **KLIPY API**.

## Features

- Browse trending GIFs from KLIPY
- Send GIFs directly into chat
- Supports Facebook / Messenger chat workflow
- Fast extension UI built with React
- Powered by the KLIPY API

## Tech Stack

- React
- Vite
- Chrome Extension Manifest V3
- Axios
- KLIPY API

## How It Works

The extension loads media from the KLIPY API and lets users pick content from the extension UI.  
Selected GIFs can then be inserted or attached into supported chat interfaces.
